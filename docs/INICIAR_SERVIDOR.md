# INICIAR SERVIDOR PARA PREVIEW

## Pasos para ver el preview en Cursor:

1. **Abre la terminal integrada de Cursor:**
   - Presiona `` Ctrl+` `` (backtick) o `Cmd+` ` en Mac
   - O ve a: View → Terminal

2. **Ejecuta este comando:**
   ```bash
   npm run dev
   ```

3. **Espera a que veas este mensaje:**
   ```
   ✓ Ready in Xs
   ○ Local:        http://localhost:3000
   ```

4. **En la pestaña Browser de Cursor:**
   - Haz click en "Enter URL"
   - Escribe: `http://localhost:3000`
   - Presiona Enter

5. **¡Listo!** Deberías ver la aplicación

## Si no funciona:

- Verifica que el puerto 3000 no esté en uso
- Cierra otros servidores de Next.js que puedan estar corriendo
- Revisa la terminal por errores
