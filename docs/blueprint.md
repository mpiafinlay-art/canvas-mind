# **App Name**: CanvasMind

## Core Features:

- Visual Canvas: Provides an infinite canvas for users to freely capture and connect ideas.
- Note Organization with AI Assistance: Integrates Genkit to offer AI-powered suggestions for organizing, summarizing, and rewriting content within notes, serving as an AI tool to enhance productivity.
- Real-time Collaboration: Allows multiple users to collaborate on the same canvas in real-time.
- Multimedia Integration: Supports embedding images from URLs and local uploads (future).
- Database and Authentication: Uses Firebase (Firestore and Firebase Authentication) to manage user data and authentication.
- Voice Dictation: Enables voice recognition to transcribe speech into text within the canvas elements.

## Style Guidelines:

- Primary color: Soft blue (#91B6EB) for primary interface elements.
- Background color: Light desaturated blue (#E2EDF9) to reduce visual fatigue.
- Accent color: Muted purple (#B882D9) to highlight key interactive elements.
- Headline font: 'Space Grotesk' (sans-serif) for titles and headings; provides a techy, accessible feel.
- Body font: 'Poppins' (sans-serif) for a versatile and modern look; consider Inter (sans-serif) for longer text for maximum neutrality.
- Special font: 'The Girl Next Door' (handwriting-style) exclusively for StickyNoteElement for a personal touch.
- Use 'lucide-react' library, which features minimalist and consistent icons.
- Clean and spacious layout with rounded corners and soft shadows for a professional and tactile feel.
